
1. Parsing

English: Parsing is the process of analyzing code or data and converting it into a format that a computer can understand and process.

ქართული: Parsing არის კოდის ან მონაცემების ანალიზისა და დამუშავების პროცესი, რომლის შედეგადაც ისინი კომპიუტერისთვის გასაგებ ფორმატში გადაიქცევა.

2. Memory Heap

English: The memory heap is a region of memory where JavaScript stores objects, arrays, and other reference data, while a program is running.

ქართული: Memory Heap არის მეხსიერების ნაწილი, სადაც JavaScript პროგრამის მუშაობისას ინახავს ობიექტებს, მასივებს და სხვა reference ტიპის მონაცემებს.

3. Hoisting

English: Hoisting is JavaScript's behavior of moving variable and function declarations to the top of their scope before the code is executed.

ქართული: Hoisting არის JavaScript-ის მექანიზმი, რომლის დროსაც ცვლადებისა და ფუნქციების დეკლარაციები თავიანთი scope-ის დასაწყისში გადადის კოდის შესრულებამდე.

4. Callback

English: A callback is a function that is passed as an argument to another function and is executed later.

ქართული: Callback არის ფუნქცია, რომელიც სხვა ფუნქციას არგუმენტად გადაეცემა და საჭირო დროს სრულდება.

5. Closure

English: A closure is a JavaScript feature, where an inner function remembers variables from its outer function even after the outer function has finished executing.

ქართული: Closure არის JavaScript-ის შესაძლებლობა, რომლის დროსაც შიდა ფუნქცია იმახსოვრებს გარე ფუნქციის ცვლადებს მისი დასრულების შემდეგაც.

6. Higher-Order Function (HOF)

English: A Higher-Order Function is a function that takes another function as an argument, returns a function, or both.

ქართული: Higher-Order Function (HOF) არის ფუნქცია, რომელიც სხვა ფუნქციას იღებს არგუმენტად, აბრუნებს ფუნქციას ან ორივეს აკეთებს.

7. Promise

English: A Promise is an object that represents the eventual completion or failure of an asynchronous operation.

ქართული: Promise არის ობიექტი, რომელიც წარმოადგენს ასინქრონული ოპერაციის მომავალ შედეგს — წარმატებას ან შეცდომას.

8. Destructuring

English: Destructuring is a JavaScript feature, that extracts values from arrays or properties from objects into separate variables.

ქართული: Destructuring არის JavaScript-ის შესაძლებლობა, რომელიც საშუალებას გვაძლევს მასივიდან ან ობიექტიდან მონაცემები პირდაპირ ცვლადებში გადავიტანოთ.

9. Error Handling

English: Error handling is the process of detecting and managing errors so a program can continue running safely.

ქართული: Error Handling არის შეცდომების აღმოჩენისა და დამუშავების პროცესი, რათა პროგრამამ სწორად გააგრძელოს მუშაობა.

10. Parsing

English: Parsing is the process of analyzing code or data and converting it into a format, that a computer can understand and process.

ქართული: Parsing არის კოდის ან მონაცემების ანალიზისა და დამუშავების პროცესი, რომლის შედეგადაც ისინი კომპიუტერისთვის გასაგებ ფორმატში გადაიქცევა.
